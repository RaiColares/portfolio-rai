## 📌 Sobre o Projeto

Este projeto consiste em um portfólio pessoal desenvolvido como requisito da formação em Desenvolvimento Web da Alpha EdTech.

O objetivo é apresentar minha trajetória profissional, habilidades técnicas e projetos desenvolvidos para clientes reais, utilizando exclusivamente tecnologias fundamentais da web:

* HTML5
* CSS3
* JavaScript Vanilla

O projeto foi construído manualmente e com ajuda de IA (Gemini), sem a utilização de frameworks de interface ou estilização, priorizando boas práticas de desenvolvimento, responsividade, acessibilidade e organização de código.

## 🔧 Como Executar Localmente

Clone o repositório:

```bash

```

Entre na pasta:

```bash
cd portfolio-rai
```

Abra o arquivo:

```bash
index.html
```

ou utilize a extensão Live Server do VS Code.
