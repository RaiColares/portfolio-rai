# 📌 Sobre o Projeto

Este projeto consiste em um portfólio profissional desenvolvido como requisito da formação em Desenvolvimento Web da Alpha EdTech.

O objetivo principal é apresentar minha trajetória profissional, habilidades técnicas e projetos desenvolvidos para clientes reais, utilizando exclusivamente tecnologias fundamentais da web.

O projeto foi construído manualmente com HTML5, CSS3 e JavaScript Vanilla, sem utilização de frameworks de interface ou estilização, priorizando:

- Código limpo e organizado
- Responsividade
- Boas práticas de desenvolvimento
- Acessibilidade
- Performance
- Experiência do usuário

# 🔧 Como Executar Localmente

Clone o repositório:

```bash
git clone https://github.com/RaiColares/portfolio-rai.git
```

Acesse a pasta:

```bash
cd portfolio-rai
```

Abra o arquivo:

```bash
index.html
```

ou utilize a extensão Live Server do VS Code.
